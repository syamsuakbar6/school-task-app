# School Task App

A task management application for school projects and assignments.

## Project Structure

- **backend/** - Backend API server
- **frontend/** - Frontend application
- **docs/** - Documentation

## Getting Started

### Prerequisites
- Docker and Docker Compose

### Installation

1. Clone the repository
2. Configure environment variables in `.env`
3. Run the application:

```bash
docker-compose up
```
